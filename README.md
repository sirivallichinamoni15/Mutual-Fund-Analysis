# Mutual Fund Analysis

## Overview

This project analyzes mutual fund performance using Python, Pandas, Matplotlib, and Power BI.

## Objectives

* Analyze fund returns and performance
* Study SIP inflows and AUM trends
* Compare mutual funds against benchmark indices
* Visualize investor and portfolio behavior

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Power BI
